# EV-Performance-Analytics-and-Visualisation-Tool-using-Python

This project analyzes and visualizes electric vehicle (EV) performance data using Python.

## Project Overview
The tool monitors key EV performance parameters such as speed, battery state of charge (SOC), power consumption, and distance travelled. Data visualization techniques are used to understand vehicle behaviour and energy usage patterns.

## Features
- EV speed analysis over time
- Battery State of Charge (SOC) monitoring
- Power consumption visualization
- Distance travelled analysis
- EV efficiency calculation (km per kWh)
- Performance comparison graphs

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Output
The project generates multiple graphs including:
- Speed vs Time
- Battery SOC vs Time
- Power Consumption vs Time
- Distance Travelled vs Time
- EV Efficiency analysis

## Applications
This type of analysis is useful for understanding EV performance trends and energy efficiency in electric vehicles.
